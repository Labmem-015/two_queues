# # About the project

Implementation of processing two queues. It's better to be lockfree. Coroutines are not allowed.

# How to build

You need to generate a project using cmake buildsystem and then build the project. You can use this command to build release version:

```
cmake --workflow release
```

Or this command to build debug:

```
cmake --workflow debug
```
